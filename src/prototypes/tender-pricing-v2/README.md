## البيانات التجريبية

### المشروع التجريبي

- **الاسم:** مشروع إنشاء مبنى إداري - الرياض
- **العميل:** وزارة الشؤون البلدية والقروية
- **عدد البنود:** 20 بند متنوع

### البنود المسعّرة مسبقاً

- **البند 1:** أعمال الحفر والردم (مُسعّر بالكامل)
- **البند 2:** صب خرسانة القواعد (مُسعّر بالكامل)
- **البنود الأخرى:** غير مُسعّرة (جاهزة للتجربة)

## المقارنة مع النظام الحالي

| الميزة          | النظام الحالي      | النموذج المحسّن v2.0             |
| --------------- | ------------------ | -------------------------------- |
| **محدد البنود** | قائمة منسدلة بسيطة | بحث + فلترة + معاينة             |
| **البحث**       | غير متوفر          | بحث فوري متقدم                   |
| **الفلترة**     | غير متوفرة         | 3 فلاتر (الكل/مُسعّر/غير مُسعّر) |
| **التنقل**      | نقرتان للانتقال    | نقرة واحدة + اختصارات            |
| **نسخ الموارد** | غير متوفر          | نسخ ولصق وتكرار سريع             |
| **الألوان**     | 5+ ألوان           | 3 ألوان رئيسية                   |
| **التباعد**     | padding: 8px       | padding: 16px (+100%)            |
| **حجم الأرقام** | 14-16px            | 20-24px (+40%)                   |
| **مؤشر الحفظ**  | غير واضح           | واضح وملون                       |
| **شريط التقدم** | بسيط               | تفصيلي مع إحصائيات               |
